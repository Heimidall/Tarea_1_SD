Para ejecutar el primer ejercio, accerder a la carpeta "Ejercicio1", luego abrir 2 terminales en esta carpeta.

En la primera terminal ejecutar el comando:    python Server.py 
En la segunda terminal ejecutar el comando:    python cliente.py

es importante hacerlo en dicho orden para el correcto funcionamiento de la tarea.



Para ejecutar el segundo ejercicio, es necesario tener instalado Docker.
Para ejecutar este ejercicio, se debe accerder a la carpeta Ejercicio2/Intento2 y desde esta carpeta,
abrir una terminal y 

ejecutar el comando:                           docker-compose build 
luego se debe ejecutar el comando:             docker-compose up
